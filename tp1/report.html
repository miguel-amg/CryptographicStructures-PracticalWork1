O problema presente no execicio 1a a implementao de uma comunicao cliente-servidor utilizando o package asyncio. Para a implementao do execicio foram criados dois ficheiros python individuais, um para o receiver e outro para o emitter.

Emitter

Para o emitter o funcionamento passa por numa primeira fase abrir coneco ao servidor atrves de um ip e de uma porta que se encontram hardcoded.

Utilizao do emitter:

$ python3 emitter.py chave

Se a ligao for estabelicida ento o emitter entra em um modo de envio de mensagens onde possivel inserir no terminal a mensagem a ser enviada para o servidor, caso contrario apresentado um erro no terminal que indica que a ligao no foi estabelecida.

Enviar mensagem: 

$ [CLIENT] Enviar mensagem para o servidor: inserir mensagem

Assim que uma mensagem enviada pelo emitter, imediatamente a seguir ele aguarda pela resposta do servidor.

Receiver

Para o receiver o funcionamento passa por comear o servio no ip e porta que se encontram hardcoded, ficando a servir por tempo ilimitado. O servidor suporta multiplos clientes e responde a cada um de forma individual.

Quando o servidor recebe uma mensagem de um cliente ele exibe no terminal a mensagem recebida com as informaes do cliente, e envia como resposta exatamente a mesma mensagem que foi recebida. Se o cliente fechar a comunicao o servidor continua a servir no mesmo ip/porta e aguarda por novos clientes. Utilizao do emitter:

Utilizao do receiver:

$ python3 emitter.py chave

Exercicio 1b

Para o exercicio 1b pretende-se implementar uma cifra AEAD utilizando o shake-256 em modo XOFHash.

Para a resoluo deste execicio foi criado o ficheiro AEAD.py que contm a implementao da cifra em uma classe.

O modelo sponge utilizado para implementar a cifra AEAD foi o seguinte

Neste modelo o rate um IV, ou seja, um nounce gerado aleatoriamente.

A capacidade do modelo a chave, em que metade dos bits da chave so o coeficiente de segurana.

Absorve

A fase de absorve funciona da seguinte forma:

O primeiro passo que executado pelo sponge a concatenao do rate da capacidade formando o estado da cifra, esse estado passado pela funo f (shake-256) que ir fornecer como output uma hash.

A partir dos primeiros rate bits do estado obtido aps a aplicao da funo F, possvel extrair um novo rate.

Com esse novo rate, realizado um OTP (One-Time Pad) com um bloco de rate bits do Associated Data (AD). Para garantir que todo o AD possa ser dividido em blocos de tamanho igual ao rate, este passa por um processo de padding adequado.

Os blocos do AD precisam ter exatamente o tamanho do rate, pois cada um deles ser combinado com o rate atravs do OTP. Durante esse processo, a capacidade (capacity) do estado permanece inalterada.

Aps a aplicao do OTP entre o plaintext e o rate, o resultado concatenado capacidade, formando assim um novo estado. Esse estado ento submetido novamente aplicao da funo F, gerando um novo valor de hash.

Este processo repete-se iterativamente para cada bloco do Associated Data, garantindo que toda a informao seja incorporada de forma consistente no estado do sponge function.

Padding do plaintext e associated data

Implementmos dois tipos de padding: PKCS7 e Zero Padding.

O tipo de padding a ser utilizado pode ser selecionado conforme necessrio.

PKCS7

O PKCS7 cria um padding que faz com que o input seja multiplo do tamanho de um bloco, o que uma propriedade util para o contexto da implementao do sponge.

O PCKS7 adiciona bytes de padding que indicam quantos bytes foram inseridos, o que permite uma remoo fcil do padding.

No entanto a vantagem do pkcs7 mais importante que ele permite detetar corrupo ou modificao dos dados, isto porque, se os dados forem truncados possivel detetar.

Ou seja este padding seguro e permite descobrir no processo de decifragem se os dados foram modificados sem a necessidade de verificar as tags de cifragem e decifragem.

No entanto no remove a necessidade da comparao de tags, apenas fornece uma segurana extra que util.

Zero Padding

Com o zero padding inserido um bit 1 no final do texto e so adicionados bits 0 at que o tamanho pretendido seja obtido

INFO_KEY = b"info-chave"
GEN_KEY_SIZE = 30
ASSOCIATED_DATA = "associated-data"

# Pode ser modificado
INFO_NONCE = b"info-nonce"
padding_mode = "pkcs" # Modos explicados abaixo (Tem de ser equivalente de ambos os lados)

# Valores possiveis para o padding_mode:
#  pkcs - Permite deteo de modificao da mensagem no recetor da mensagem)

# Lidar com uma mensagen recebida
async def recMsg(reader, writer):
    # Instanciar AEAD
    aead_cipher = AEAD.aead()

    # Obter a chave
    shared_key = sys.argv[1]
    shared_key_bytes = shared_key.encode()

    # Gerar a key
    aead_key = kdf.derive_key(shared_key_bytes, GEN_KEY_SIZE, SALT, INFO_KEY)
    print("[RECEIVER] Chave gerada.")
    aead_nounce = kdf.derive_key(shared_key_bytes, GEN_KEY_SIZE, SALT, INFO_NONCE)
    print("[RECEIVER] Nounce gerado.")

    # Receber comunicao do cliente
    addr = writer.get_extra_info('peername')
    print(f"[RECEIVER] Conexo estabelecida com {addr!r}")

    # Receber mensagens
    while True:
        data = await reader.read(100) # Esperar por mensagem
    
        if not data:
            break  # Coneccao do cliente desligada

        # Modo debug
        if(view_cyphertexts):
            print(f"[RECEIVER-DEBUG] Ciphertext recebido: {data}")

        originaltext_bytes, decrypt_tag = aead_cipher.decrypt(data, ASSOCIATED_DATA, aead_key, padding_mode)
        originaltext = originaltext_bytes.decode() # Passar o plaintext para texto
        print(f"[RECEIVER] Mensagem decifrada: {originaltext!r} enviada por {addr!r}.")

        # Visualizar a tag caso necessrio
        if(view_tags):
            print(f"[RECEIVER-DEBUG] Tag da mensagem decifrada: {decrypt_tag!r} enviada por {addr!r}.")

        # Enviar a mesma mensagem de volta
        ciphertext_send, encrypt_tag = aead_cipher.encrypt(originaltext, ASSOCIATED_DATA, aead_key, aead_nounce, padding_mode)
        print(f"[RECEIVER] Mensagem espelhada enviada (automtico): {originaltext!r}")
        writer.write(data)
        await writer.drain()

        # Modo debug
        if(view_cyphertexts):
            print(f"[RECEIVER-DEBUG] Ciphertext enviado: {ciphertext_send}")
        if(view_tags):
            print(f"[RECEIVER-DEBUG] Tag da mensagem enviada: {encrypt_tag}")

    print(f"[RECEIVER] Conexo fechada com {addr!r}")
    writer.close()
    await writer.wait_closed()

async def main():
    # Verificar se a chave foi recebida como argumento

    async with server:
        await server.serve_forever()

from cryptography.hazmat.primitives.kdf.hkdf import HKDF
from cryptography.hazmat.primitives import hashes

# Funo que cria deriva uma chave utilizando o algoritmo HKDF com hash SHA256
def derive_key(master_key: bytes, length: int, salt: bytes, info: bytes) -> bytes:
    # Verificaes
    if not master_key:
        raise ValueError("A chave no pode estar vazia.")
    if not salt:
        raise ValueError("O salt no pode estar vazio.")
    if not info:
        raise ValueError("O info no pode estar vazio.")

    # Advertir caso a chave seja demasiado pequena
    if len(master_key) < 16:
        print("[KDF] Aviso: A chave tem menos de 16 bytes. Entropia insuficiente para derivar uma chave segura.")

    # O HKDF j estende a chave de forma determinstica para o tamanho desejado
    hkdf = HKDF(
        algorithm=hashes.SHA256(),
        length=length,
        salt=salt,
        info=info,
    )
    return hkdf.derive(master_key)

# Imports
import sys
import secrets
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives import padding

#############################################
# Valores alteraveis (hardcoded)
NOUNCESIZE = 16 # Tamanho do nounce default
#############################################

# Classe que contm o codigo da cifra aead (Tipos de padding: zero ou pkcs) 
# Utiliza zero padding por default
class aead:
    # A cifra vai receber a mensagem (plaintext), dados associados (ad), uma chave (key)
    def encrypt(self, plaintext="", ad="", key_bytes=b'', iv=None, paddingType="zero"):
        # Se no for passado um IV,  gerado um nonce default
        if iv is None:
            iv = secrets.token_bytes(NOUNCESIZE)

        # Verificao do input
        self._checkdata(plaintext, ad, key_bytes)

        # Preparao das variaveis
        plaintext_bytes = plaintext.encode() 
        ad_bytes = ad.encode() 
        blockSize = len(iv) # Tamanho do bloco

        # Preparao (Adicionar padding)
        ad_padded = b''
        plaintext_padded = b''
        if (paddingType == "zero"):
            ad_padded = self.__AddPaddingZero(ad_bytes, blockSize) 
            plaintext_padded = self.__AddPaddingZero(plaintext_bytes, blockSize)
        elif (paddingType == "pkcs"):
            ad_padded = self.__AddPaddingPKCS(ad_bytes, blockSize) 
            plaintext_padded = self.__AddPaddingPKCS(plaintext_bytes, blockSize)
        else:
            raise ValueError("[AEAD] Tipo de padding desconhecido.")

        # Execuo do algoritmo de cifragem
        absorve_state = self.__absorve(iv, key_bytes, ad_padded) # Realizar o absorve
        result = self.__squeeze(plaintext_padded, absorve_state, blockSize, key_bytes, mode="encrypt")

        # Cria um campo de 2 bytes com o tamanho do nonce
        nonce_length = len(iv).to_bytes(2, byteorder="big")

        # Resultado concatenado: [tamanho do nonce][nonce][ciphertext]
        return nonce_length + iv + result[0], result[1]

    # A cifra vai receber a mensagem (ciphertext), dados associados (ad), uma chave (key)
    def decrypt(self, ciphertext_bytes=b'', ad="", key_bytes=b'', paddingType="zero"):
        # Extrair os primeiros 2 bytes para obter o tamanho do nonce
        nonce_length = int.from_bytes(ciphertext_bytes[:2], byteorder="big")

        # Extrair o nonce
        iv = ciphertext_bytes[2:2+nonce_length]
        
        # Extrair o ciphertext
        ciphertext_bytes = ciphertext_bytes[2+nonce_length:]

        # Verificao do input
        self._checkdata(ciphertext_bytes, ad, key_bytes)

        # Preparao das variaveis
        ad_bytes = ad.encode() 
        blockSize = len(iv)

        # Preparao (Adicionar padding)
        ad_padded = b''
        if (paddingType == "zero"):
            ad_padded = self.__AddPaddingZero(ad_bytes, blockSize)
        elif (paddingType == "pkcs"):
            ad_padded = self.__AddPaddingPKCS(ad_bytes, blockSize)
        else:
            raise ValueError("Tipo de padding desconhecido.")

        # Execuo do algoritmo de cifragem
        absorve_state = self.__absorve(iv, key_bytes, ad_padded) # Realizar o absorve
        plaintext_padded, tag = self.__squeeze(ciphertext_bytes, absorve_state, blockSize, key_bytes, mode="decrypt")

        # Remover o padding da cifragem
        unpadded_plaintext= b''
        if (paddingType == "zero"):
            unpadded_plaintext = self.__RemovePaddingZero(plaintext_padded, blockSize)
        elif (paddingType == "pkcs"):
            unpadded_plaintext = self.__RemovePaddingPKCS(plaintext_padded, blockSize)
        
        return unpadded_plaintext, tag

    # Mtodo privado que adiciona padding a uma sequncia de bytes (usando pkcs7) 
    # - Faz com que o tamanho do input seja multiplo do tamanho do bloco
    # - Permite detetar ataques de comprimento 
    # - Permite remoo fcil do padding. 
    def __AddPaddingPKCS(self, input: bytes, blockSize: int) -> bytes:
        padder = padding.PKCS7(blockSize * 8).padder() # So trabalha com bits o pkcs7
        padded = padder.update(input) + padder.finalize()
        return padded

    # Mtodo privado que remove padding
    def __RemovePaddingPKCS(self, input_data: bytes, blockSize: int) -> bytes:
        unpadder = padding.PKCS7(blockSize * 8).unpadder()
        try:
            unpadded = unpadder.update(input_data) + unpadder.finalize()
        except ValueError as e:
            print("[AEAD] O algoritmo de padding PKCS7 detetou corrupo de dados.")
            sys.exit(1)  # Parar o algoritmo
        return unpadded

    # Metodo privado de zero padding
    def __AddPaddingZero(self, input_data: bytes, blockSize: int) -> bytes:
        remainder = len(input_data) % blockSize
        if remainder == 0:
            padding_size = blockSize
        else:
            padding_size = blockSize - remainder
        padding = b'\x01' + (b'\x00' * (padding_size - 1))
        return input_data + padding

    # Metodo privado de remoo de zero padding
    def __RemovePaddingZero(self, input_data: bytes, blockSize: int) -> bytes:
        if not input_data:
            raise ValueError("[Padding] Erro: Dados vazios.")
        last_padding_index = input_data.rfind(b'\x01') # Procurar pelo bit de inicio de padding apartir do lado direito
        if last_padding_index == -1 or last_padding_index < len(input_data) - blockSize:
            raise ValueError("[Padding] Erro: Padding invlido ou corrompido.")
        return input_data[:last_padding_index] # Extrair todos os bytes at ao bit de inicio do padding

    # Metodo privado que realiza o absorve do sponge (Devolve o ultimo estado)
    def __absorve(self, starting_rate, key_bytes, padded_data):
        # Preparao para o absorve
        ad = padded_data
        blockSize = len(starting_rate)
        cycles = int(len(padded_data) / blockSize) 
        state = starting_rate + key_bytes
        state_size = len(state)

        # Iteraes do absorve
        for n in range(cycles):
            # Aplicar a funo F (shake-256)
            digest = hashes.Hash(hashes.SHAKE256(digest_size=state_size))
            digest.update(state) # Dar como input o estado atual
            state = digest.finalize() # Receber o output que passa a ser o proximo estado

            # Preparao para o OTP
            new_rate = state[:blockSize] # Extrair os primeiros rate bits do estado (rate do estado atual)
            state    = state[blockSize:] # Remover os primeiros rate bits do estado (criar espao para o resultado)
            block    = ad[:blockSize]    # Extrair os primeiros rate bits dos metadados (bloco de dados do ad)
            ad       = ad[blockSize:]    # Remover os primeiros rate bits dos metadados (para iterar o ad)

            # Aplicar OTP
            xor_result = bytes(x ^ y for x, y in zip(new_rate, block)) # Realizar 'rate XOR metadados'
            state = xor_result + state
        
        return state

    # Metodo privado que realiza o squeeze do sponge
    # input_data: O que vai sendo inserido ao longo do squeeze
    def __squeeze(self, input_data, state, blockSize, key_bytes, mode="encrypt"):
        extracted_data = b''  # Resultado do squeeze
        cycles = int(len(input_data) / blockSize)  # Nmero de iteraes
        state_size = len(state)  # Tamanho do state

        # Iteraes normais do squeeze
        for n in range(cycles):
            rate        = state[:blockSize]      # Extrair os primeiros rate bits do estado (rate do estado atual)
            state       = state[blockSize:]      # Remover os primeiros rate bits do estado (rate do estado atual)
            input_block = input_data[:blockSize] # Extrair um bloco do input_data (ciphertext ou plaintext)
            input_data  = input_data[blockSize:] # Remover um bloco do input_data

            # Calcular a sada (XOR entre rate e input_block)
            xor_result = bytes(x ^ y for x, y in zip(rate, input_block))
            extracted_data += xor_result

            # Atualizao do state: (Vamos utilizar o ciphertext em ambos para os estados estarem sincronizados)
            if mode == "encrypt":
                state = xor_result + state # Ao fazer cifragem, o bloco usado  o xor_result (que  o ciphertext)
            else:
                state = input_block + state # Ao fazer decifragem, o bloco usado  o input_block (o ciphertext recebido)

            # Atualizar o state (SHAKE256)
            digest = hashes.Hash(hashes.SHAKE256(digest_size=state_size))
            digest.update(state)
            state = digest.finalize()

        # Processar a ltima iterao (key-mixing e tag)
        if cycles > 0:
            rate = state[:blockSize]             # Extrair os primeiros rate bits do estado (rate do estado atual)
            state = state[blockSize:]            # Remover os primeiros rate bits do estado (rate do estado atual)
            input_block = input_data[:blockSize] # Extrair um bloco do input_data (ciphertext ou plaintext)
            input_data = input_data[blockSize:]  # Remover um bloco do input_data

            xor_result = bytes(x ^ y for x, y in zip(rate, input_block))
            extracted_data += xor_result

            # Atualizao do state: (Vamos utilizar o ciphertext em ambos para os estados estarem sincronizados)
            if mode == "encrypt":
                state = xor_result + state # Ao fazer cifragem, o bloco usado  o xor_result (que  o ciphertext)
            else:
                state = input_block + state # Ao fazer decifragem, o bloco usado  o input_block (o ciphertext recebido)

            # Atualizar o state (SHAKE256)
            digest = hashes.Hash(hashes.SHAKE256(digest_size=state_size))
            digest.update(state)
            state = digest.finalize()

        # A tag  gerada fazendo XOR final do state com a chave
        key_for_tag = (key_bytes * ((state_size // len(key_bytes)) + 1))[:state_size]
        tag = bytes(x ^ y for x, y in zip(state, key_for_tag))
        return extracted_data, tag

    # Metodo privado que verifica os dados recebidos
    def _checkdata(self, plaintext, ad, key_bytes):
        if not plaintext:
            raise ValueError("Erro: O `plaintext` no pode estar vazio.")
        if not ad:
            raise ValueError("Erro: O `ad` (Associated Data) no pode estar vazio.")
        if not key_bytes:
            raise ValueError("Erro: A `key` (chave) no pode estar vazia.")
